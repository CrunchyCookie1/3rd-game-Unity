= How to add the project to unity =

After accepting the invite to join the GitHub repo you can now do the following steps

Download the github desktop app:
https://desktop.github.com/download/

The version of unity that is being used is Version 6000.3.9f1
If the version isn't in the install unity editor section then you can go to the Archive. Here's the link: https://unity.com/releases/editor/archive
You may need to click on LTS to see the 6000.3.9f1 version.

on GitHub desktop app
- go to file
- clone repository
- then find the 3rd Game Unity Repository
(Remember the location for where the files are being saved. This can be seen before cloning in the local path section)

Now on the Unity Hub
- go to the drop down for open
- click on add project from disk
- find the project in the location that you saved it in
(normally this is in documents/GitHub/3rd game unity)
- click on the project folder and add it to your unity hub

= How to save changes made by you =

after you've made a change like say adding a cube you first save the project then.

- go to the GitHub desktop app
(you'll see all the changes you've made)
- you then add a title in the bottom left (usually)
- you can also add in a description of what changes you've made
- after all that press the commit to main button
- Now press the push origin button
